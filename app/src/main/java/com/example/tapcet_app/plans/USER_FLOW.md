# User Flow

This document outlines the user's journey through the Nemeno Quiz App.

## 1. Onboarding Flow

1.  **App Launch**: User opens the app for the first time and sees a splash screen.
2.  **Sign Up / Log In**: The user is prompted to create an account or log in.
3.  **Welcome**: A brief welcome message is displayed.

## 2. Main Learning Loop

1.  **Dashboard**: The user lands on the main dashboard, which displays a curated list of courses (e.g., "Algebra", "Biology", "History").
2.  **Course Selection**: The user selects a course to begin.
3.  **Subject Selection**: Inside a course, the user sees a list of subjects or topics.
4.  **Quiz Start**: The user chooses a quiz to start.
5.  **Quiz Screen**: The user is presented with questions one by one.
    *   They select an answer for each question.
    *   Immediate feedback can be provided (correct/incorrect).
6.  **Results Screen**: After the quiz is complete, the user sees a summary of their performance, including their score and the number of correct/incorrect answers.
7.  **Return to Dashboard**: The user returns to the dashboard, where their progress for the completed quiz and course is updated.

## 3. Profile and Progress

1.  **Access Profile**: The user can navigate to their profile from the dashboard.
2.  **View Stats**: The profile screen displays statistics like points earned, current streak, and completed courses.
3.  **Achievements**: The user can view any badges or achievements they have unlocked.
