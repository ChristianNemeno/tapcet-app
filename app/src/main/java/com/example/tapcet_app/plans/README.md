# Nemeno Quiz App

## Project Overview

Nemeno is a mobile quiz application designed to help high school students prepare for college entrance exams. It provides a gamified learning experience, similar to Duolingo, with a curated dashboard of courses and subjects.

## Features

*   **Curated Course Dashboard**: A guided path through high school subjects.
*   **Subject-Specific Quizzes**: Quizzes focused on specific topics within a subject.
*   **Gamified Learning**: Points, streaks, and levels to motivate users.
*   **User Progress Tracking**: Visual indicators of user progress in each course.
*   **User Profile**: A section for users to view their stats and achievements.

## Tech Stack

*   **Platform**: Android
*   **Language**: Kotlin
*   **UI**: Jetpack Compose
*   **IDE**: Android Studio
*   **Build Tool**: Gradle
